# iOS Home Screen Installation Instructions

## Quick Method (iOS Safari):

1. **Upload the files to a web host:**
   - fibula-donor-selector.html
   - manifest.json
   - icon-192.png
   - icon-512.png

2. **Open Safari on your iPhone/iPad**
   - Navigate to the URL where you uploaded the HTML file

3. **Add to Home Screen:**
   - Tap the Share button (square with arrow pointing up)
   - Scroll down and tap "Add to Home Screen"
   - Customize the name if desired (default: "Fibula Selector")
   - Tap "Add" in the top right

4. **Done!**
   - The app icon will appear on your home screen
   - Tap it to launch the app in full-screen mode

## Alternative: Use GitHub Pages (Free Hosting)

1. Create a GitHub account (if you don't have one)
2. Create a new repository called "fibula-selector"
3. Upload all four files to the repository
4. Go to Settings → Pages
5. Enable GitHub Pages from the main branch
6. Your app will be live at: `https://yourusername.github.io/fibula-selector/fibula-donor-selector.html`
7. Open that URL in Safari and follow steps 3-4 above

## Alternative: Use iCloud Drive

1. Save all four files to iCloud Drive in a folder
2. On your iPhone, open the Files app
3. Navigate to the HTML file
4. Tap and hold the file → Share → Open in Safari
5. Follow steps 3-4 from Quick Method above

Note: The app will work offline once added to your home screen for the current session, but requires an initial internet connection to load.
